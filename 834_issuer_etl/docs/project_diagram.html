3. Processing modes

flowchart TD
    START[python main.py] --> MODE{PROCESSING_MODE}
    MODE -->|local| LOC[scan source_data only]
    MODE -->|sftp| AUDIT{SFTP_AUDIT_ONLY}
    AUDIT -->|true| SCAN[recursive audit to CSV]
    AUDIT -->|false| DL[download flatten then pipeline]
    LOC --> FULL[parse reconcile validate report assets]
    DL --> FULL
    SCAN --> END[exit no parser]

4. Pipeline stages

flowchart LR

    B --> C[2 Parse and load SQLite]
    C --> D[3 Reconcile]
    D --> E[4 Validate]
    E --> F[5 Reports]
    F --> G[6 Assets export]

5. Project folder map (text)
